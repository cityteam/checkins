Backup Database

When you are through with the checkins for a particular day (and after you have printed the summary report, if needed), click on the Admin and then Database Backup links in the top level navigation menu.

Then, click on the button. It only takes a few seconds to run.

Once the backup completes, you will see the following completion report. If you see anything else, such as an alert with error messages, please take a picture of it with your phone, and email it to your appropriate technical support representative.

This operation copies the entire database information to a file that is local to wherever the application has been installed. Your administration person will need to make sure this file gets copied off (typically to a thumb drive) so that nothing will be lost even if the application's server crashes and cannot be recovered.
