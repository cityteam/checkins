Adjust Existing Checkin

Things are looking great! We have checked in everyone waiting so far, and can continue doing so until we are full or run out of people waiting for a mat.

But now, Barney remembers that he really does not really need to shower until 6. How do we change the existing information?

Simply click on Barney's row (mat 2), and we will see what our options are.

For an existing assignment, we can do one of three different things:

  1. Edit the existing assignment details. (we will do this here)
  2. Move the Guest to a different mat. (we will show this next)
  3. Remove the assignment completely. (a last resort to erase a mistake)
Or, if we got here by mistake, simply click the button to exit with no changes.

In this particular case, simply change the existing shower time (05:00:00) to 06:00:00 instead, and click the button to store this change.

While we are making sure that Barney's change got reflected, Fred looks up and sees his best friend is alredy here. Would it be possible for Fred to switch mats and sleep next to Barney overnight?

Of course it is! Click on Fred's row (mat 4) to see how.

This time, we are adjusting the current mat 4 assignment to Fred (as the title confirms). Next to the To Mat: prompt, use the dropdown to select which mat you want to transfer Fred to (it will only offer the available ones, 1 and 3), then click the button to perform this reassignment.

If you have a previously assigned Guest that decides not to stay, you can erase their assignment by clicking on the existing row and clicking the button. You will then need to confirm the removal by clicking the button on the warning page that pops up.

After you have finished adding or editing assignments for the evening, and printed the summary page (if needed), there is one more task for you to perform.

Follow the Backup Database link below for more information.
