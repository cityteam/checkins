Starting A Day

Click on the at the top of the screen. This will transfer you to the place where you will do nearly all of your work.

The first question you must answer is the date on which you want to perform checkins. Since this is almost always today's date, it will conveniently default to that value. Simply press Enter to pick the date currently shown. If you want to go back to a previous date (for example, to see what was done yesterday), update the date first and then press Enter.

The very first time you access a particular date, there will be no mats available to assign to Guests. You must select from the templates that have been set up by an administrator for your Facility - ask if you are not sure which one to use.

Once you have selected the desired template, the will become active, and you can click it.

Now, we see the result of generating mats from a template that defines four available mats. The actual set of mat numbers is part of the template definition, so how many there are will change depending on which template you selected.

But what do those letters after the mat numbers mean? Again, the template can define optional features for each particular mat as follows (a mat with no special features will not have any suffix):

  • H - specially equipped for handicap access.
  • S - near an electrical socket.
  • W - Work mat, normally reserved for candidates for the recovery program (if available).

Now, we are ready to start assigning Guests for tonight to particular mats.
