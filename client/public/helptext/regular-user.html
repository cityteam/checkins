Logging In
When you first start the application, you will see this banner.

To log in, click the button at the top, enter your assigned username and password, and click the button at the bottom.

If you make a mistake in either the username or the password, an error alert will pop up. If that happens, simply clear it, fix the error, and click the button again.

NOTE: We are using a test Facility for the purposes of these instructions, in order to not disturb the information for any actual Facility.

Now the application knows who you are, and what CityTeam Facilities you have permissions to interact with. If you can interact with more than one Facility, use the Facility: dropdown in the upper right corner to switch, at any time.

Now, you can use any of the options in the navigation bar across the top to choose what you want to do. See the following pages for the usual next steps.
